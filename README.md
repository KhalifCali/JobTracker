
#  Job Tracker – Fullstack Application

A modern fullstack application for tracking job applications, managing statuses and visualizing progress.


## 🌐 Live Demo

 *(coming soon)*

##  Overview

This project is a fullstack system where users can:

* Track job applications
* Update application status (Applied, Interview, Rejected, Offer)
* Filter and manage jobs dynamically
* View analytics of job progress
* (Coming) Secure login with JWT authentication



## ⚙️ Tech Stack

**Frontend**

* React
* TypeScript

**Backend**

* ASP.NET Core (.NET)
* Entity Framework

**Database**

* SQL Server / In-Memory (current)

**Other**

* REST API
* Azure (deployment in progress)



## 🔥 Key Features

* Full CRUD API for job management
* Filtering jobs by status
* Clean separation between frontend and backend
* Dynamic UI updates using React state
* Scalable backend structure using ASP.NET Core



## 🏗️ Architecture


React (Frontend)
        ↓
ASP.NET Core API
        ↓
Database (SQL / In-Memory)


## ▶️ Run Locally

### Frontend

cd jobtracker-frontend
